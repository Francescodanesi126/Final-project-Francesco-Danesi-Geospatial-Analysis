# Final-project-Francesco-Danesi-Geospatial-Analysis
Spatial analysis of cancer mortality and income in Lombardy
## How to Reproduce
1. Install dependencies:
pip install -r requirements.txt

2. Open `notebooks/analysis.ipynb`
3. Run all cells

## Main Findings
- Cancer mortality exhibits significant negative spatial autocorrelation.
- Income per capita does not show significant spatial clustering.
- No statistically significant relationship between income and mortality was found.
- Spatial structure remains in regression residuals.
